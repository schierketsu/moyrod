# СВОИКОРНИ.РФ

СВОИКОРНИ:  пет-проект — веб-приложение для родословной. Фронт на **Vue 3 + Vite**, бэк — один **Express** с **SQLite** (`better-sqlite3`). Карточки людей, связи, поле/карта, аккаунты и JWT, подсказки по населённым пунктам через внешние API (Photon / Open-Meteo). 

## Запуск локально

Из корня репозитория:

```bash
npm install
npm run dev
```

Поднимаются **Vite** (по умолчанию [http://localhost:5173](http://localhost:5173)) и **API** на порту **3456**; запросы с фронта на `/api` проксируются в Express (см. `vite.config.js`).

Только сервер (например, после сборки статики):

```bash
npm run server
```

Сборка фронта и запуск как прод (отдаётся `dist/`, если есть):

```bash
npm run build
npm start
```

База создаётся сама при первом старте: `data/svoi-korni.sqlite` (папку `data/` можно не коммитить, если не хочешь делиться дампом).

Для продакшена задай свой секрет JWT, иначе остаётся дефолт из кода:

```bash
set JWT_SECRET=длинная-случайная-строка
```

(в PowerShell: `$env:JWT_SECRET="..."`.) Порт сервера: переменная `PORT` или **3456**.
